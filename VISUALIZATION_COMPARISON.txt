INPUT TEXT:
"John works at Google in California."


┌─────────────────────────────────────────────────────────────────────┐
│ 1. ENTITY VISUALIZATION (langextract)                               │
│    Location: outputs/entity_visualizations/                         │
│    Purpose: Shows WHERE entities appear in original text            │
└─────────────────────────────────────────────────────────────────────┘

   "[John] works at [Google] in [California]."
     ^^^^           ^^^^^^      ^^^^^^^^^^
     │              │           └─ Location (animated)
     │              └─ Organization (animated)
     └─ Person (animated)

   Features:
   • Entities highlighted with colors
   • Animated sequential highlighting
   • Shows original text context
   • Legend for entity types
   • NO relations shown


┌─────────────────────────────────────────────────────────────────────┐
│ 2. GRAPH VISUALIZATION (Plotly + NetworkX)                          │
│    Location: outputs/graph_visualizations/                          │
│    Purpose: Shows RELATIONSHIPS between entities                    │
└─────────────────────────────────────────────────────────────────────┘

        ┌──────────┐
        │   John   │ (Person)
        └─────┬────┘
              │ works at
              ↓
        ┌──────────┐
        │  Google  │ (Organization)
        └─────┬────┘
              │ located in
              ↓
        ┌──────────────┐
        │  California  │ (Location)
        └──────────────┘

   Features:
   • Entities as nodes
   • Relations as edges (arrows)
   • Interactive network (zoom/pan)
   • Hover for details
   • Shows graph structure
   • NO original text

│ Text-based               │ Network-based                            │

TOGETHER THEY PROVIDE COMPLETE INSIGHT:
✅ Entity viz: "John" appears at position 0-4 in the text
✅ Graph viz: "John" is connected to "Google" via "works at"


╔══════════════════════════════════════════════════════════════════════╗
║                     TECHNICAL DETAILS                                ║
╚══════════════════════════════════════════════════════════════════════╝

ENTITY VISUALIZATION:
  Triples → Find spans in text → langextract.AnnotatedDocument
  → langextract.visualize() → Animated HTML

GRAPH VISUALIZATION:
  Triples → NetworkX graph → GraphML → Plotly → Interactive HTML


BOTH ARE CREATED AUTOMATICALLY BY THE PIPELINE!
